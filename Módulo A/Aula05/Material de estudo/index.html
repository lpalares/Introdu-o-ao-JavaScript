Comentários

Comentários servem, como o próprio nome diz, para comentar nos códigos. Seja para explicar a linha ou bloco ou seja para organizar o código, comentários são extremamente úteis (ainda mais se você trabalhar com uma equipe). No JavaScript existem duas formas de comentar no código. A primeira forma é usando // e a segunda forma é com /* */. O comentário com // serve apenas para uma linha enquanto com /* */ serve para colocar quantas linhas de comentário forem necessárias. O que estiver dentro dos comentários não será executado.

Exemplos

Usando //:
//Seu comentário de linha única.

Usando /* */
/*
Seu comentário com diversas linhas.
Você pode usar quantas linhas quiser,
desde que esteja dentro de /* */.
*/

O que são variáveis?

Variáveis são como caixas onde você pode guardar seus pertences. Na programação, as variáveis armazenam informações que seu site, jogo ou projeto necessitam para depois serem utilizadas ou manipuladas. No entanto, variáveis simples conseguem armazenar apenas uma informação por vez, sendo necessário tirar o valor antigo para guardar o valor novo. Cada variável precisa de um identificador (nome) para serem encontradas. Nunca coloque o mesmo nome para variáveis diferentes.

Regras para identificadores

  1. Podem começar com letra, $ ou _
  2. Não podem começar com números
  3. É possível usar letras ou números
  4. É possível usar acentos e símbolos
  5. Não podem conter espaços
  6. Não podem ser palavras de comandos do JavaScript

Dicas

  1. Maiúsculas e minúsculas fazem diferença
  2. Tente escolher nomes coerentes para variáveis
  3. Evite se tornar um "programador alfabeto" ou um "programador contador"

Tipos primitivos

Os tipos primitivos são classificações dadas para cada informação que você pode armazenar em uma variável. Veja alguns exemplos:

Número

São números idade, peso, altura e etc. No JavaScript números inteiros e números reais ficam na mesma categoria enquanto nas outras linguagens não.

Inteiros

1, 5, 10, 9, 7, 11...

Reais ou float

1.05, 8.5, 10.35...

NaN

Uma variável se torna NaN (Not a Number ou "não é um número") quando passa a armazenar um valor numérico indefinido ou impossível de calcular. Isso ocorre principalmente após operações matemáticas inválidas ou tentativas de conversão incorretas.

Strings

Strings são uma cadeia de caracteres. Esse tipo pode ter números, mas costumam ter mais coisas como pontuação e letras

"Maria", "José", "(11) 99999-9999"...

Boolean ou booleano

Valores booleanos são apenas dois: true e false.

Outros tipos

Infinity (tipo dentro do tipo número)

Null

Undefined

Object

Array (tipo dentro do tipo object)
