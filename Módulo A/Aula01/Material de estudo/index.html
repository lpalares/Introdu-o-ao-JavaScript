Introdução

No desenvolvimento web é necessário usar HTML, CSS e alguma linguagem de programação, como o JavaScript. HTML cria o conteúdo, CSS cria a aparência e o JavaScript é o sistema de engrenagens que faz a mágica por trás da interatividade e as funções importantes do site funcionarem.

Quem usa?

Basicamente a internet inteira usa o JavaScript! Essa é uma linguagem de programação bastante popular entre os desenvolvedores de sites. Empresas famosas usam JavaScript em seus sites. Alguns exemplos são: Youtube, Google, LinkedIn, Netflix e Facebook. Inclusive, sem o JavaScript, o Youtube não funciona.
