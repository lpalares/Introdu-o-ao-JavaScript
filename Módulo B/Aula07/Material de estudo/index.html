Operadores Aritméticos

Operadores aritméticos são usados para fazer cálculos. Existem seis operadores no JavaScript, sendo eles:

+

É usado para adição.

-

É usado para subtração.

*

É usado para multiplicação.

/

É usado para divisão com resultado em valores reais.

%

É usado para pegar o resto da divisão com resultado em valores inteiros.

**

É usado para indicar a potência.

Ordem de precedência

  1. ()
  2. **
  3. * / %
  4. + -

Operadores de incremento e decremento

Adiciona ou subtrai 1. Use ++ para adicionar 1 à variável e -- para subtrair 1.

Exemplo

            let x = 10;
            x++;

Operadores de atribuição

Operadores que atribuem valores.

=

Atribuição simples. Atribui valor.

+=

Soma e atribui.

-=

Subtrai e atribui

*=

Multiplica e atribui.

/=

Divide e atribui.

**=

Potencializa pelo operando e atribui.

%=

Divide e atribui o resto da divisão.

Operadores relacionais

São operadores que comparam dois valores.

>

Maior que. Compara se valor da esquerda é maior que o valor da direita.

<

Menor que. Compara se valor da esquerda é menor que o valor da direita.

>=

Maior ou igual que. Compara se valor da esquerda é maior ou igual ao da direita.

<=

Maior ou igual que. Compara se valor da esquerda é maior ou igual ao da direita.

==

Compara se os valores são iguais.

!=

Compara se valores são diferentes.

Operadores de identidade

Eles comparam se dois valores são idênticos em valor e em tipo de dado, sem realizar conversões implícitas.

===

Estritamente igual. Retorna true se ambos os operandos forem do mesmo tipo e possuírem o mesmo valor.

!==

Estritamente igual. Retorna true se ambos os operandos não forem do mesmo tipo e possuírem o mesmo valor.

Operadores lógicos

!

É uma negação. Significa "não".

&&

É uma conjunção. Significa "e".

||

São disjunções. Significa "ou".

Ordem de precedência

  1. !
  2. &&
  3. ||

Operador ternário

Operador ternário tem esse nome porque possui três operandos ou blocos. No operador ternário é usado ? e : dentro da mesma expressão.

O primeiro bloco, antes de ?, é um teste lógico. Se o teste lógico for verdadeiro, será executado o bloco do meio (entre ? e :), agora se for falso, será executado o último bloco (depois de :).

Exemplo

média >= 7.0 ? "Aprovado" : "Reprovado"

Ordem de precedência dos operadores

  1. Operadores aritméticos
  2. Operadores relacionais
  3. Operadores lógicos
